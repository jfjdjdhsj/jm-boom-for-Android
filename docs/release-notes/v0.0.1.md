# v0.0.1

## 功能

- 新增 Android 首版发布流程，支持 `arm64-v8a`、`armeabi-v7a`、`x86_64` 和 `x86` 四种架构 APK。
- 保留现有首页、搜索、排行、收藏、历史、下载、设置、详情页和阅读器功能。

## 优化

- 适配 Android 手机布局，导航改为底部触控栏，列表、详情页、设置页和阅读器调整为响应式布局。
- Android 更新检查改为读取当前仓库 GitHub Release，并在发现新版本时打开下载页。

## 修复

- 修正 Android 版本使用的 Release 仓库地址，避免更新检查指向原始桌面仓库。

## 其他

- Release APK 文件名格式为 `JM-Boom-架构-v版本号.apk`。
- Android APK 由 GitHub Actions 构建、签名并自动上传到 Release，本地不执行 APK 构建。
