# v0.0.2

## 功能

- Android 启动时请求“所有文件访问权限”，用于读写外部公共目录。
- Android 下载文件改为保存到 `/storage/emulated/0/jj-boom/downloads`。
- Android 运行日志改为保存到 `/storage/emulated/0/jj-boom/logs/jm-boom.log`。

## 优化

- Android 页面统一适配状态栏、手势导航栏和安全区，改善全面屏设备显示效果。
- 增强手机与平板响应式布局，平板横屏和桌面宽度使用更合适的导航与内容布局。
- 阅读器底栏在手机端重新排布，避免页码与阅读设置按钮重叠。
- 设置页缓存区不再显示内部缓存路径和打开文件夹按钮，仅保留缓存大小、上限和清理入口。

## 修复

- 修复 Android 设置页打开日志目录时可能出现 `No such file or directory` 的问题。
- 修复 Android 首页全面屏手势返回时未返回桌面的行为。

## 其他

- GitHub Actions 在初始化 Tauri Android 工程后自动注入外部存储权限和自定义 `MainActivity`。
- 本版本仍由 GitHub Actions 构建分架构 APK，本地不执行 APK 构建。
